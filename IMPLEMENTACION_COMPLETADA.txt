## 🔧 Características Técnicas

### Distribución de Columnas Por Ancho

```
MUY CORTAS (35-40px):  id, correcta
CORTAS (50px):         periodo, grado, imagen  
MEDIANAS (80-90px):    evaluacion, area, id_contexto
LARGAS (180-220px):    contexto, enunciado, opcion_a, opcion_b, opcion_c, opcion_d
```

### Sistema de Control

```
Grid Layout (2x2):
┌──────────────────────────────┬──┐
│   TreeView (14 columnas)     │ │ Scroll Vertical
│                              │ │
├──────────────────────────────┤ │
│     Scroll Horizontal        │ │
└──────────────────────────────┴──┘
```

### Redimensionamiento

```
Parámetro: stretch=True
Permite:
- Arrastrar bordes de encabezado
- Mantener proporciones
- Sin límite de tamaño mínimo (salvo 0px)
```

```

```

## ⚙️ Configuración Actual

```python
# Parámetros del TreeView
height = 15                    # Filas visibles
show = "headings"             # Mostrar encabezados
columns = 14                  # Total de columnas
total_width = 1625px          # Ancho de contenido

# Scrollbars
vertical = habilitado         # Para navegar filas
horizontal = habilitado       # Para navegar columnas

# Redimensionamiento
stretch = True                # Usuario puede cambiar ajustes
```

## 🚨 Notas Importantes

1. **Configuración de Columnas NO se Guarda**
   - Los cambios de ancho se pierden al cerrar la app
   - Esto es por diseño (estado transiente)

2. **Orden de Columnas NO es Reordenable**
   - Están fijas: id, evaluacion, area, ... imagen
   - No se puede reordenar arrastrando

3. **Altura del TreeView es Fija**

## 📞 Soporte

### Si No Ves el Scroll Horizontal

**Causa:** Las columnas caben en tu pantalla  
**Solución:**
1. Reduce el ancho de la ventana, O
2. Amplía manualmente algunas columnas grandes, O
3. Aumenta la resolución establecida en los combos de filtros

### Si el Texto Sigue Cortado

**Causa:** Ancho de columna insuficiente  
**Solución:**
1. Ubica el borde del encabezado (cursor vinculado: ↔)
2. Arrastra hacia la derecha
3. Suelta cuando sea adecuado

### Si Necesitas Restaurar Valores Predeterminados

**Solución:**
1. Cierra la aplicación completamente
2. Reabre - los anchos se resetean automáticamente
