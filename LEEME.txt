────────────────────────────────────────
Elige UNA de las siguientes opciones:

   OPCIÓN A (Automática - Recomendada):

   a) Abre PowerShell en el directorio del proyecto
   b) Ejecuta: python integrar_banco_preguntas.py
   c) Responde "s" cuando pregunte
   d) ¡Listo! El archivo modulo_superadmin.py fue actualizado

   OPCIÓN B (Manual - Si prefieres control):
   ─────────────────────────────────────────
   a) Abre modulo_superadmin.py en VS Code
   b) Busca "def _build_preguntas_tab(self):"
   c) Reemplaza TODO ese método con:
   
      def _build_preguntas_tab(self):
          from interfaz_banco_preguntas import InterfazBancoPreguntasAvanzada
          InterfazBancoPreguntasAvanzada._build_preguntas_tab_mejorada(self)
   
   d) Guarda el archivo (Ctrl+S)


PASO 4: Ejecutar la aplicación
──────────────────────────────
Corre tu aplicación normalmente:

    python app.py  # o lo que uses

Abre SuperAdmin > "Banco Preguntas" > ¡Disfruta! 🎉

🎯 EJEMPLO DE USO MÍNIMO
═════════════════════════════════════════════════════════════════════════════════

from banco_preguntas_profesional import BancoPreguntasProfesional

# Crear instancia
banco = BancoPreguntasProfesional("preguntas.xlsx")

# 1. Cargar todas las preguntas
todas = banco.obtener_todas_preguntas()
print(f"Total: {len(todas)} preguntas")

# 2. Filtrar por grado
grado5 = banco.obtener_preguntas_filtradas(grado="5")
print(f"Grado 5: {len(grado5)} preguntas")

# 3. Obtener una pregunta
pregunta = banco.obtener_pregunta_por_id("P001")
if pregunta:
    print(f"Enunciado: {pregunta['enunciado']}")

# 4. Guardar nueva
datos_nueva = {
    'id': 'P999',
    'evaluacion': 'Trimestral',
    'area': 'Matemáticas',
    'grado': '5',
    'enunciado': '¿Cuánto es 3+3?',
    'opcion_a': '5',
    'opcion_b': '6',
    'opcion_c': '7',
    'opcion_d': '8',
    'correcta': 'B'
}
exitoso, msg = banco.guardar_pregunta('P999', datos_nueva, es_nueva=True)
print(f"Guardado: {msg}")

# 5. Importar masivamente
resumen = banco.importar_masivo('importar.xlsx')
print(f"Importadas: {resumen['exitosas']}")

# 6. Validar integridad
advs = banco.validar_integridad()
if not any(advs.values()):
    print("✓ Banco íntegro")

🆘 SOLUCIÓN DE PROBLEMAS
═════════════════════════════════════════════════════════════════════════════════

PROBLEMA: ModuleNotFoundError: No module named 'pandas'
SOLUCIÓN: pip install pandas openpyxl

PROBLEMA: Los archivos no se guardan en Excel
SOLUCIÓN: Verifica que el archivo no esté abierto en Excel

PROBLEMA: "Columnas faltantes" al importar
SOLUCIÓN: Tu archivo Excel no tiene todas las 14 columnas esperadas

PROBLEMA: "El ID 'X' ya existe"
SOLUCIÓN: Cambia el ID a uno único o usa Excel para verificar duplicados

PROBLEMA: El TreeView no muestra preguntas
SOLUCIÓN: Verifica que preguntas.xlsx exista y tenga datos

PROBLEMA: ImportError en integración automática
SOLUCIÓN: Asegúrate de que banco_preguntas_profesional.py esté en el mismo
          directorio que modulo_superadmin.py
