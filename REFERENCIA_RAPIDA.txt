# REFERENCIA RÁPIDA - Mejoras TreeView Banco de Preguntas

## 📌 Cambios en Una Página

### El Problema
```
ANTES: 7 columnas solo | Sin scroll horiz. | Anchos desproporcionados
AHORA: 14 columnas todas | Con scrolls ambos | Anchos inteligentes
```

### Dónde Ver los Cambios
```
Archivo:   interfaz_banco_preguntas.py
Líneas:    110-190 (TreeView) + 260-295 (Carga de datos)
Cambios:   Ampliación + Scrolls + Anchos optimizados
```

### Lo Nuevo
1. **Scroll Horizontal** ← Navega por todas las 14 columnas
2. **Scroll Vertical** ← Más preguntas (mejorado)
3. **Redimensionamiento** ← Arrastra bordes de encabezados
4. **14 Columnas** ← Todas visibles

### Cómo Usarlo
```
Ver opciones completas:
└─ Scroll horizontal → opcion_a | opcion_b | opcion_c | opcion_d | correcta

Ampliar columna:
└─ Arrastra borde encabezado → más ancho/menos ancho

Filtrar por grado:
└─ Combo superior → selecciona grado
```

### Preguntas Comunes
```
P: ¿Cómo navego horizontalmente?
R: Usa la barra de scroll en la parte inferior o Shift+Rueda

P: ¿Puedo cambiar el ancho de columnas?
R: Sí, arrastra el borde del encabezado

P: ¿Se guardan los cambios de ancho?
R: No, se resetean al cerrar la aplicación

P: ¿Funciona con mis datos de Excel actuales?
R: Sí, 100% compatible

P: ¿Necesito instalar algo nuevo?
R: No, solo usa lo que ya tienes (tkinter)
```
