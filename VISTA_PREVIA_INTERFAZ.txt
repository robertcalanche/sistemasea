FLUJO DE INTERACCIÓN - CASOS DE USO:
═════════════════════════════════════════════════════════════════════════════════

CASO 1: VER TODAS LAS PREGUNTAS
────────────────────────────────
1. Abre SuperAdmin
2. Pestaña "Banco Preguntas"
3. ✓ TreeView carga automáticamente TODAS las preguntas
4. Muestra estadísticas: Total: 245 | Grados: 7 | Áreas: 5...


CASO 2: FILTRAR POR GRADO Y ÁREA
─────────────────────────────────
1. Click en combo "Grado" → Selecciona "5"
   ✓ Combo "Área" se actualiza automáticamente
2. Click en combo "Área" → Selecciona "Matemáticas"
   ✓ Combo "Evaluación" se actualiza automáticamente
3. Click en combo "Evaluación" → Selecciona "Trimestral"
   ✓ TreeView muestra solo las 5 preguntas filtradas
4. Estadísticas se actualizan en tiempo real


CASO 3: AGREGAR NUEVA PREGUNTA
───────────────────────────────
1. Click botón "➕ Agregar"
2. Se abre diálogo "Pregunta"
3. Completa todos los campos (14 totales, 9 requeridos)
4. Click "Guardar"
5. ✓ Validaciones automáticas:
   - ¿ID es único?
   - ¿Todos los campos requeridos tienen datos?
   - ¿Opción correcta es A/B/C/D?
6. Si todo OK → Se guarda en Excel y TreeView se actualiza
7. Si error → Mensaje rojo explicando qué falta


CASO 4: EDITAR PREGUNTA EXISTENTE
────────────────────────────────
1. Selecciona una fila en el TreeView
2. Click botón "✏️ Editar"
3. Se abre diálogo con los datos actuales precargados
4. Modifica lo que necesites
5. Click "Guardar"
6. ✓ Se valida (excepto ID, que permanece igual)
7. Excel se actualiza y TreeView refresca


CASO 5: ELIMINAR PREGUNTA
──────────────────────────
1. Selecciona una fila en el TreeView
2. Click botón "🗑️ Eliminar"
3. Pregunta de confirmación: "¿Está seguro de eliminar la pregunta ID P001?"
4. Click "Sí"
5. ✓ Se elimina de Excel, TreeView refresca, reindexación automática


CASO 6: IMPORTAR 100 PREGUNTAS MASIVAMENTE
───────────────────────────────────────────
1. Prepara archivo Excel: importar.xlsx con estructura completa
2. Click botón "📥 Importar masivo"
3. Selecciona archivo
4. El sistema procesa cada fila:
   ✓ Valida que no exista el ID
   ✓ Valida que no exista el enunciado
   ✓ Valida que todos los campos requeridos existan
   ✓ Valida que opción correcta sea A/B/C/D
5. Se muestra resumen:
   - Importadas: 95 ✓
   - Duplicadas ID: 3 ⚠
   - Enunciados duplicados: 1 ⚠
   - Rechazadas validación: 1 ✗
6. Se guardan las 95 exitosas en Excel
7. TreeView se actualiza con los nuevos datos


CASO 7: EXPORTAR PREGUNTAS FILTRADAS
─────────────────────────────────────
1. Aplica filtros (ej: Grado 5, Área Matemáticas)
2. TreeView muestra solo 20 preguntas
3. Click botón "📤 Exportar"
4. Selecciona ubicación para guardar
5. ✓ Se genera reporte_grado5.xlsx con solo esas 20 preguntas
6. Mensaje: "Exportadas 20 preguntas a: C:\...\reporte_grado5.xlsx"


CASO 8: VALIDAR INTEGRIDAD EL BANCO
───────────────────────────────────
1. Click botón "✓ Validar integridad"
2. Se abre ventana con resultados:
   - IDs duplicados (si existen)
   - Enunciados duplicados (si existen)
   - Campos vacíos (si existen)
   - Opciones correctas inválidas (si existen)
3. Si no hay problemas: ✓ "No se encontraron problemas"
4. Si hay problemas: Muestra lista detallada

DIÁGNOSTICO VISUAL - ¿Cómo sé si funciona correctamente?
═════════════════════════════════════════════════════════════════════════════════

✓ Signos de funcionamiento correcto:
  □ TreeView carga con preguntas al abrir la pestaña
  □ Filtros se actualizan dinámicamente
  □ Botones responden al click
  □ Los cambios se guardan en Excel
  □ Mensajes de validación aparecen cuando corresponde
  □ Importación muestra resumen detallado

✗ Signos de problema:
  □ TreeView vacío al abrir
  □ Filtros no cambian Área/Evaluación
  □ Los cambios no se guardan
  □ Error "ModuleNotFoundError"
  □ El archivo Excel se corrompe

→ Ver troubleshooting en README_BANCO_PREGUNTAS.md
